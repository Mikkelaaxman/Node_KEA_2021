Callbacks

Are "return statements" that happens after another function has completed. In JavaScript, functions are objects. Because of this, functions can take functions as arguments, and can be returned by other functions. Functions that do this are called higher-order functions. Any function that is passed as an argument is called a callback function. Because JavaScript is event-driven, it wont wait for some slow API call to execute the next function and if ordering of the functions isn't upheld, we need callbacks to make sure certain code doesnt execute before it is time.
