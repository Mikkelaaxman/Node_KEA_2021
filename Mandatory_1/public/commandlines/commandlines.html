Useful Terminal Commands

Windows CMD and Powershell

List of CMD commands

  • cd *filename or tab* to go down folder
  • cd .. to go up
  • dir to see files in directory
  • mkdir *name* to create a new folder
  • Ctrl+C to stop running job

    • NPM

  • npm init attempts to auto create new package.json
  • npm install to install dependencies already in package file
  • npm run *script* run code with script from package file like start-dev
  • npm i -g *package* install package globally
  • npm un *package* uninstall package
  • npm ls lists installed packages

    • Global Environment

    Setting windows environment variables can be useful e.g. to choose a port
